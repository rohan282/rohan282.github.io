Projects

OpportunityOdyssey

I developed OpportunityOdyssey as a comprehensive job portal using Spring Boot 3, implementing role-based access control to serve both job seekers and recruiters effectively. The platform features advanced job search filters and resume upload functionality, enabling seamless job applications and candidate management.

The system includes robust features for job posting and application tracking, allowing recruiters to view detailed applicant information and manage their hiring pipeline efficiently. I utilized Hibernate/JPA for reliable data persistence coupled with MySQL for optimized data storage, ensuring scalable performance.

To create a smooth user experience, I integrated Thymeleaf for server-side rendering and implemented Spring Security for comprehensive authentication and authorization, making the platform both user-friendly and secure.

QuickCart

QuickCart represents my exploration into RESTful API development, where I built a comprehensive e-commerce solution using Spring Data REST with automated endpoint generation for seamless products, orders, and user management. The architecture focuses on efficiency and scalability through intelligent automation.

I implemented sophisticated database relationships, including complex many-to-many mappings between orders and products with proper cascade operations to maintain data integrity. The entity models feature comprehensive validation, bidirectional relationships, and custom field mappings designed for optimal data handling performance.

The project leverages Maven for streamlined dependency management and Apache Tomcat for reliable deployment, while providing comprehensive CRUD operations through automated REST endpoint generation, creating a robust foundation for e-commerce functionality.
